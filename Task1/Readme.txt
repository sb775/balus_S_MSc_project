This folder contains:

1) Code for formating and grouping alleles from VDJbase according to gene names. File name: 'task1_create_fasta.py'

2) An input file called 'VDJbase_dataset130521.fasta'.

3) A folder called 'fasta' containing the 25 allelic sequences described in section 2.1 and Table 1.

4) The output file called 'summary_table'

5) R code called 'Task1_prettymsa.r', used to create the MSA (seen in Figure A, Appendix). 

6) A folder called 'msa_pretty_aln' containing MSA files of all the IGHV gene groups from Table 1.
